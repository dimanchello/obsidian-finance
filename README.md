# 💰 Finance Tracker — плагин для Obsidian  (v2)

Учёт доходов и расходов прямо в заметках Obsidian.  
Каждая заметка — отдельный **счёт** (наличные, карта, крипто-кошелёк и т.д.).

---

## Установка

```bash
# Скопируйте папку в .obsidian/plugins/finance-tracker/
npm install
npm run build
```

Настройки Obsidian → Сторонние плагины → включить **Finance Tracker**.

---

## Использование

Вставьте в любую заметку:

````markdown
```finance-account
```
````

## Возможности v2

### Счёт
| | |
|---|---|
| **Название** | Кликните на заголовок → редактируется inline |
| **Валюта** | Кликните на бейдж рядом с названием → выбор из списка или своя |

### Форма записи
- Обязательное поле — только **Сумма**
- Поля **Дата** и **Время** рядом
- **Умное автозаполнение**: при вводе категории или плательщика  
  автоматически подставляются сумма, тег и второе поле из последней  
  совпадающей записи (появляется бейдж «✨ данные подставлены»)
- **Примечание** — визуально выделено цветом акцента

### Таблица
- На десктопе — полноценная таблица с закреплённым заголовком
- На мобильном/планшете — каждая запись отображается как карточка  
  с подписями полей (адаптивный card-view)

### Импорт / Экспорт (кнопка ⇅)

**Экспорт:**
- Форматы: CSV, JSON, XML
- Выгружаются все записи текущего счёта

**Импорт:**
1. Выберите файл (.csv / .json / .xml)
2. Для JSON — укажите путь к массиву, если он не в корне (напр. `data.records`)
3. Для XML — укажите тег одной записи (автоопределяется)
4. Настройте соответствие полей файла ↔ полям счёта
5. Укажите как определять тип: по полю, по знаку суммы, или все доходы/расходы
6. Нажмите «Импортировать»
